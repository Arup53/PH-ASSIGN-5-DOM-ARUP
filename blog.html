Important FAQ

What is the Document Object Model (DOM)?

Technically, Document Object Model is a data structure, particularly a tree structure, that a web browser creates when it receives an HTML text. The browser provides some API's to modify this structure and the results is shown live.

How do you select an element from the DOM?

To select a DOM elements, there are various methods. Some of them are give below

  • 1. .getElementById() : only select single elements
  • 2. .querySelector() : only select single elements
  • 3. .getElementByClassName() : select all elements under that class name
  • 4. .getElementsByTagName() : select all elements under that tag name
